Тема: переменные

- В JavaScript можно объявлять переменные для хранения данных. Это делается при помощи var. - Можно записать значение и без объявления переменной, однако по ряду причин это не рекомендуется. - Вместе с объявлением можно присвоить значение, например: var x = 10. - Можно заменять и копировать значение переменной. - Переменные, которые названы БОЛЬШИМИ_БУКВАМИ, являются константами, то есть никогда не меняются. Существуют целые нотации для обозначения переменных. CamelCase, lowerCamelCase, snake_case и т.д.
